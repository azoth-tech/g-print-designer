# AI API Configuration

## OpenAI (for text, layouts, elements)
# Get your API key from: https://platform.openai.com/api-keys
OPENAI_API_KEY=sk-your-openai-api-key-here

## DeepAI (for image generation)
# Get your API key from: https://deepai.org/dashboard/profile
DEEPAI_API_KEY=your-deepai-api-key-here

# For local development:
# 1. Copy this file to .env.local
# 2. Replace the API keys with your actual keys
# 3. Restart your development server

# For Cloudflare Pages deployment:
# 1. Go to your Cloudflare Pages dashboard
# 2. Navigate to Settings > Environment variables
# 3. Add both OPENAI_API_KEY and DEEPAI_API_KEY with your actual keys
